# グリーン車通勤損益計算アプリ

## これは何？

「グリーン車で通勤すれば、仕事ができる時間が増えるけど、料金も高い…結局お得なの？」

このアプリは、そんな疑問に答えます。通勤中に作業できる時間の価値と、グリーン車の追加料金を比較して、「本当に得するのか」を日・週・月・年単位で見える化します。  
`docs/spec.md` に詳細な設計書を保管しています。

## セットアップ

```bash
bun install
bun run dev
```

ブラウザで <http://localhost:3000> を開いて動作を確認してください。

## 主な機能

- 時給・通勤時間・出社頻度・グリーン車差額を入力する設定フォーム
- サーバーアクションによる損益計算と Zod + React Hook Form によるバリデーション
- 日次 / 週次 / 月次 / 年次の期間別損益表示
- 損益分岐点の時給およびコスト内訳表示
- 入力した設定値をローカルストレージに保存し、次回アクセス時に自動復元

## 技術スタック

- Next.js App Router / Server Actions
- Tailwind CSS
- React Hook Form + Zod
- Bun

## ディレクトリガイド

- `src/app` : ルーティングおよびページコンポーネント
- `src/components` : UI コンポーネント
- `src/lib` : 計算ロジックやフォーマッター
- `src/types` : 型定義
- `docs/spec.md` : 仕様書（元 README.md）
